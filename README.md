# Thanh Thái Frontend Frameworks

## Giới thiệu
Thanh Thái Frontend Frameworks là một bộ công cụ giúp các lập trình viên tại Việt Nam dễ dàng phát triển và triển khai giao diện người dùng bằng các framework phổ biến như React, Vue và Angular. Dự án này bao gồm các ví dụ, hướng dẫn và các thành phần UI tái sử dụng được.

## Tính năng
- Hướng dẫn từng bước cài đặt và sử dụng cho mỗi framework
- Bộ mẫu dự án starter cho từng framework
- Bộ công cụ UI gồm các component được tối ưu cho hiệu suất,
- Các ví dụ thực tiễn và bài tập từ cơ bản đến nâng cao

## Cài đặt
Để bắt đầu với dự án này, bạn có thể tải về mã nguồn từ repository này và làm theo hướng dẫn trong từng folder của các framework.

## Đóng góp
Hãy góp ý hoặc gửi pull request nếu bạn có ý tưởng để cải thiện dự án này! Chúng tôi luôn chào đón sự đóng góp từ cộng đồng.
